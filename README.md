# neerav-portfolio

My personal portfolio — [live site](https://neerav-portfolio-nine.vercel.app).

A single self-contained `index.html`: no framework, no build step, no external
requests. All CSS, JS, and animations (canvas network mesh, auto-typing
terminal, scroll reveals, animated metrics) are hand-written and inlined.

## Run locally

```bash
open index.html   # or just drag it into a browser
```

## Deploy

Any static host works. This repo is wired to Vercel — push to `main` and it
ships.
